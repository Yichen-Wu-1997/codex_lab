# 03_feature_tests

This stage is for small feature work with a tests-first workflow.

The current mini project already works. Your job later will be to add one bounded feature by writing tests first and then implementing only what those tests require.

## Goal

Practice asking Codex to:

- inspect the current code and tests
- propose a short plan
- add tests for the new behavior first
- implement the smallest code change that makes those tests pass

## Run the tests

```powershell
python -m unittest discover -s tests
```
