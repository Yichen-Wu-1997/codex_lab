# 04_workflow_rules

This stage is about improving the way you work with Codex, not just the code it writes.

The mini project is tiny on purpose. The real learning target is durable workflow guidance: better `AGENTS.md` rules, better review habits, and better explanations after changes.

## Goal

Practice asking Codex to:

- inspect repo guidance before editing
- propose a bounded plan
- make a small code or docs change
- explain the result clearly using a reusable structure

## Run the tests

```powershell
python -m unittest discover -s tests
```
